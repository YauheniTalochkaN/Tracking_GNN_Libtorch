# Graph Neural Network with Attention and Two-Stage Aggregation for Particle Track Reconstruction
This repository contains the LibTorch-based code of a GNN model for particle track reconstruction produced in high energy physics experiments, in particular, at the MPD of the NICA.

## Publication
The code in this repository corresponds to the following publication:

> Y. Talochka, G. Ososkov, N. Voytishin, *Graph Neural Network with Attention and Two-Stage Aggregation for Particle Track Reconstruction in the TPC MPD of the NICA Accelerator Complex*, Computing and Software for Big Science, vol. 9, p. 22, Dec. 2025.

If you use this code for your research, please cite the above paper.
